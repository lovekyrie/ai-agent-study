# 学习笔记

本目录存放项目相关的概念问答与复盘材料，便于后续查阅。

## 目录

| 路径 | 说明 |
|------|------|
| [learning/INDEX.md](./learning/INDEX.md) | 按日期索引的问答日志 |
| [typescript/](./typescript/) | TypeScript / tsconfig 专题 |
| [packages/config/](./packages/config/) | `@ai-agent-study/config` 包说明 |

## 如何新增

在 Cursor 中提问时，若启用 [learning-qa-journal](../.cursor/skills/learning-qa-journal/SKILL.md) skill，Agent 会在回答后自动追加到 `docs/learning/` 并更新索引。

也可手动在对应专题目录下补充 markdown。
