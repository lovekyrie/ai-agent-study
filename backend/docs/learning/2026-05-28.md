# 学习日志：2026-05-28

## Q1: 为什么第 6、7 步结果先打印，章节标题日志反而在后面？

**上下文**：`backend/stages/stage02-prompt-context/src/example.ts` 的 `runSection`

**要点**：

- 代码顺序：先 `logger.info` 标题，再 `await fn()` 里的 `console.log`
- 终端顺序：`console.log` 先出现，因 pino-pretty 走异步 worker；`console.log` 同步写 stdout
- 第 1、2 步同样会标题滞后，只是第 6、7 步无网络延迟更明显

**详文**：[../packages/logger/pino-console-order.md](../packages/logger/pino-console-order.md)
